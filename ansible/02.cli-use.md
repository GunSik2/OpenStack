
## Quick Start
- Inventory
```
$ sudo vi /etc/ansible/hosts

[beta]
10.10.10.3
10.10.10.9

[beta:vars]
ansible_python_interpreter=/usr/bin/python3
```

- Run command
```
$ ansible beta -m ping
10.10.10.3 | SUCCESS => {
    "changed": false,
    "ping": "pong"
}
10.10.10.9 | SUCCESS => {
    "changed": false,
    "ping": "pong"
}
```
